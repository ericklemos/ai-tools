# Discovery Document Template

Use one file per discovery in this folder.

## Filename

- Recommended: `YYYY-MM-DD-<topic>-<short-slug>.md`
- Example: `2026-03-13-debugging-jwt-clock-skew.md`

## Template

```md
# [Discovery title]

## Context
[Where and when this was found]

## Discovery
[What was learned and why it matters]

## Reuse
[How other agents should apply this]

## Evidence
[Commands, logs, or facts that support it]
```
